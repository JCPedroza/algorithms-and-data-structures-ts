# Selection Sort

## Categories

- Polynomial (cuadratic) growth
- Comparison sort
- Stable
- In-Place

## Time Complexity

### Comparisons

- Worst: `O(n²)`
- Best: `O(n²)`
- Average: `O(n²)`

### Swaps

- Worst: `O(n)`
- Best: `O(1)`
- Average: `O(n)`

## Space Complexity

- Total: `O(n)`
- Auxiliary: `O(1)`
